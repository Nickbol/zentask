Plan your time with simple and easy-to-use event management app.

Select tags

Select the sections you are interested in (you can select up to 3 priority tags)

Create tasks

Create tasks easily. You can select the priority and category of the task, set up a reminder at a certain time and it’s repeatability.

Manage tasks

View the list of created tasks by day and easily manage it: edit, delete, filter. If the task is completed, check «Do it» and it’ll move from the list of active tasks to the list of completed tasks.

Filter tasks

Filter your tasks easily for Priority, Tag, Start or End Date, Status: Active, Done, All.
